class states.levels.level.Level(level: int, music_file: str, imgArr=None)¶

General level class.

Contains all of the attributes and behavior that a level should contain.

Args:

level (int): Level number. music_file (str): Music file name for the current level. imgArr (list[str], optional): List of background image paths. Defaults

to None.

handle_events(events)¶

All input events from are handled here.

Note:
Every state that inherits from this class needs to implement the

loop to process input events. pygame.menu.Menu objects require the entire list of events to update, so this is a workaround for that for now.

Args:

events (list[pygame.event.Event]): List of events.
