utils.timer module¶

class utils.timer.Timer(start_time=0, start=False)¶

Bases: object

A helpful class to represent an in-game timer that counts up. This timer can be paused, resumed, and reset.

Args:
start_time (int, optional): The starting time of the timer in seconds.

Defaults to 0.

start (bool, optional): Whether or not the timer should start on

instantiation. Defaults to False.

Attributes:

start_time (int): The starting time of the timer in seconds. paused_time (float): The time when this timer was paused. is_running (bool): Whether the timer is currently running or not. color (str): The color of the timer’s text to display screen (pygame.Surface): The screen to draw to.
